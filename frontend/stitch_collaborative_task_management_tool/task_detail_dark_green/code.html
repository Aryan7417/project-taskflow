Implement Interactive Shader

description Description

We need to integrate a high-performance WebGL shader for the new hero section. The shader should respond to cursor movement and provide a subtle "emerald mist" effect over a dark void background to match our current design language.

Key requirements:

  • Must maintain 60fps on average hardware.
  • Use our primary color palette (Primary: #4edea3, Void: #0f1510).
  • Include fallback for older browsers or reduced motion preferences.
shader.glsl 
void main() {
    vec2 uv = gl_FragCoord.xy / u_resolution.xy;
    vec3 color = vec3(0.06, 0.72, 0.50); // Primary Emerald
    float dist = distance(uv, u_mouse);
    
    float intensity = smoothstep(0.5, 0.0, dist);
    gl_FragColor = vec4(color * intensity, 1.0);
}
